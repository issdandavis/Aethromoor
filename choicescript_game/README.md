### Option 1: Use ChoiceScript IDE (Recommended - Works Like a Real App!)

1. **Download ChoiceScript:**
   - Go to: https://www.choiceofgames.com/make-your-own-games/choicescript-intro/

   - Open `index.html` in the ChoiceScript folder
   - Click "Play"
   - **It now works like a real Choice of Games app!**

1. Go to: https://www.choiceofgames.com/cside/
2. Upload the scene files from `choicescript_game/scenes/`

## üìù CHOICESCRIPT SYNTAX GUIDE

This game uses standard ChoiceScript commands:

- `*choice` - Player choices
- `*set` - Modify variables
- `*if` / `*else` - Conditional logic
- `*goto` - Jump to labels
- `*goto_scene` - Jump to different scene file
- `*stat_chart` - Display stats
- `*achievement` - Award achievements
- `*finish` - End the game
