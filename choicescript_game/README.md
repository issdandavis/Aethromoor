## ğŸ“ CHOICESCRIPT SYNTAX GUIDE

This game uses standard ChoiceScript commands:

- `*choice` - Player choices
- `*set` - Modify variables
- `*if` / `*else` - Conditional logic
- `*goto` - Jump to labels
- `*goto_scene` - Jump to different scene file
- `*stat_chart` - Display stats
- `*achievement` - Award achievements
- `*finish` - End the game
