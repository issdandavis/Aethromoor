## ðŸ“ž Questions to Ask for Each PR

Before making a decision, answer these:

1. **What is this PR trying to do?**
2. **Is this work still relevant?**
3. **Is there a newer/better version of this change?**
4. **Will merging this break anything?**
5. **Do I actually want this feature/change?**

If you can't answer "yes" confidently to most questions, consider closing the PR.
