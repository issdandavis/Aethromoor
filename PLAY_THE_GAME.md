## üéØ ChoiceScript Version (For Publishing)

The ChoiceScript version is the professional, publishable edition of the game.

### Quick Play Steps:

1. **Install Node.js** (if not installed): https://nodejs.org/
2. **Navigate to ChoiceScript folder**:
   ```bash
   cd /path/to/Avalon/choicescript_game
   ```
3. **Install dependencies**:
   ```bash
   npm install
   ```
4. **Start the server**:
   ```bash
   node server.js
   ```
5. **Open browser** to: `http://localhost:3000`

## ‚ùì Troubleshooting

**HTML Version not working?**
- Make sure you're opening `game/index.html` not the folder
- Try a different browser (Chrome, Firefox, Edge all work)

**ChoiceScript Version not working?**
- Confirm Node.js is installed: `node --version`
- Make sure you're in the choicescript_game folder
- Check server.js exists in the folder
- Try port 8080 if 3000 is in use: `node server.js --port 8080`
