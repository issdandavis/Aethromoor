## üîß TECHNICAL NOTES

**File Structure:**
- startup.txt must load first
- All scenes in scenes/ folder
- Scene list in startup.txt defines order available to player
- *goto_scene jumps between files
- *goto jumps within same file

**Variables:**
- All 159 variables declared in startup.txt
- Use *set to modify
- Use ${variable} to display
- Use *if for conditionals

**Achievements:**
- Defined in startup.txt
- Triggered with *achievement command
- visible/hidden status
- Point values for gamification
