## ‚ö†Ô∏è Important Notes

### Campaign vs. Canon
**D&D campaign materials** (PDFs with "Campaign" in title) are **source material**, not automatically canonical. They should be:
- Mined for character moments and world details
- Curated through the Master Chronicle process
- Verified against established lore before integration

### Master Chronicle Authority
The **IZACK_MASTER_CHRONICLE_UPDATED.txt** file represents curated, canonical lore that has been:
- Extracted from campaign materials
- Refined for consistency
- Approved for narrative integration

When conflicts arise between campaign archives and the Master Chronicle, **the Master Chronicle takes precedence**.

### Document Versions
Multiple versions of documents exist for historical reference. Always use:
- Most recent date stamps
- "UPDATED" or "FINAL" designations
- Higher version numbers (e.g., "2.4" over "2.0")
